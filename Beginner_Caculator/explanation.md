# Explanation

## Step 1

Print a welcome message using:

```python
print()
```

---

## Step 2

Store information inside variables.

Example:

```python
name = input("Enter your name: ")
```

---

## Step 3

Convert text input into numbers.

```python
age = int(input("Enter your age: "))
```

or

```python
number = float(input("Enter a number: "))
```

## Step 4

Use arithmetic operators.

| Operator | Meaning |
|----------|---------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| // | Floor Division |
| % | Modulus |
| ** | Power |

## Step 5

Display formatted output using f-strings.

```python
print(f"Name: {name}")
```
