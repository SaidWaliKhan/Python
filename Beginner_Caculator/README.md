# 🐍 Python Fundamentals Playground

## Project 01

This beginner-friendly project introduces the most important Python fundamentals.

## 📚 Concepts Covered

- Variables
- Data Types
- User Input
- Output
- Arithmetic Operators
- f-Strings
- Basic Program Structure

---

## 📋 Features

### Personal Information Card

- Ask the user for:
  - Name
  - Age
  - Country
  - Favorite Programming Language

Display the information in a nicely formatted card.

---

### Basic Calculator

Ask the user for two numbers and calculate:

- Addition
- Subtraction
- Multiplication
- Division
- Floor Division
- Modulus
- Power

---

## ▶️ How to Run

```bash
python solution.py
```

---

## 🎯 Learning Outcome

After completing this project you will understand:

- Variables
- User Input
- Number Conversion
- Arithmetic Operators
- Formatted Output
- Basic Python Program Structure
